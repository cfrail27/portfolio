Sobre Mi

"Soy un desarrollador web con una pasión por crear experiencias digitales excepcionales.
Desde joven me sentí atraído por la tecnología.
Mi camino en el desarrollo web comenzó hace 4 años, cuando descubrí el poder de la tecnología para transformar ideas en productos funcionales y atractivos."

"Comencé a aprender desarrollo web de manera autodidacta, y después de completar estudios formales en Grado Superior en Desarrollo de Aplicaciones Web, decidí enfocarme más en el desarrollo Front-End.
Me apasiona transformar ideas en productos tangibles, con una interfaz moderna, responsiva y accesible."

"Creo firmemente que la clave para desarrollar software de calidad está en la colaboración y la comunicación. Me esfuerzo por escribir código limpio, mantenerme actualizado con las últimas tecnologías y siempre poner al usuario en el centro de mis decisiones."

"Me especializo en tecnologías como HTML, CSS y JavaScript.
Estoy constantemente aprendiendo nuevas tecnologías para estar al tanto de las mejores prácticas en la industria.
En la actualidad me encuentro estudiando el framework Angular con mayor profundidad."

"Uno de los proyectos de los que más me enorgullezco es El Proyecto Fin de Grado, una aplicación web para compra y venta de artículos de magia.
Este proyecto me permitió implementar mis conocimientos en HTML, CSS, JavaScript, PHP y SQL.
Fué una experiencia enriquecedora en la que aprendí mucho."

"Fuera del trabajo, me encanta el deporte y el maravilloso mundo de la magia.
Creo que es importante tener un balance entre lo profesional y lo personal para mantener la creatividad y la energía."

Mis Proyectos

Aventura Marítima - Sitio Web de Alquiler de Embarcaciones

En Aventura Marítima, trabajo en el desarrollo de la parte Front End de la página principal de una web dedicada al alquiler de embarcaciones.
Las tecnologías empleadas para su desarrollo han sido HTML y CSS.
Los requisitos que debía cumplir son un diseño responsive, así como la utilización de flex layout y grid layout.

Carrito de compra - Aplicación web de un carrito de compra de un e-commerce

En el proyecto Carrito de Compra, se pide implementar la funcionalidad de un carrito de compra de un e-commerce.
Las tecnologías utilizadas para desarrollar este proyecto han sido HTML, CSS y Javascript.
No se pide que sea un diseño responsive, por tanto realizamos la versión para PC.

Explosión de Sabores - Sitio Web de Recetas Gastronómicas

Explosión de Sabores se centra en el desarrollo Front End de la página web principal de un portal de recetas culinarias.
Para ello he hecho uso de las tecnologías HTML, CSS y Javascript.
En esta ocasión se exige que el diseño sea responsive, así como la utilización de css grid layout y css flexbox.

El Garaje de César - Sitio Web de un Blog sobre el mundo del Motor

El Garaje de César es una web destinada a ser un blog sobre el mundo del motor. Me piden trabajar en la parte Front End de la página principal del sitio web.
Uno de los requisitos es que en la página principal se vean un par de entradas del blog, así como un diseño responsive del mismo.
En cuanto al footer no se me dan indicaciones.

Moda Madrid - Sitio Web de Moda Masculina y Femenina

En Moda Madrid desarrollo la parte Front End de la página principal de una web de moda.
Se exige que el diseño sea responsive, y que en la página principal haya un grid con diferentes columnas y filas mostrando modelos de ropa masculina y femenina.
No se me exige ninguna indicación para el footer.

Proyecto Fin de Grado - Sitio Web Tienda Online de Artículos de Magia

El proyecto fin de grado ha consistido en la realización de una tienda online.
Con este proyecto, he puesto en práctica los conocimientos adquiridos durante el estudio del Grado Superior en Desarrollo de Aplicaciones Web.
He utilizado todo lo referente a creación de aplicaciones web y operaciones CRUD mediante el uso de HTML, CSS, Bootstrap, PHP, SQL y JavaScript.
